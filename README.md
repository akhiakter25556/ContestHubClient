= Features (10+ Bullet Points)
1. User can register and log in securely (JWT-based authentication)
2. Live contest list with detailed contest pages
3. Big banner image & full contest description section
4. Real-time countdown timer for contest deadline
5. Secure payment system to join contest
6. After successful payment, participant count increases automatically
7. Creator/admin can declare a winner
8. Winner name + photo shown after declaration
9. Contest status updates automatically (Live → Ended)
10. Fully responsive UI for mobile, tablet, desktop
11. User dashboard to track registered contests
12. Admin dashboard to manage contests and participants
